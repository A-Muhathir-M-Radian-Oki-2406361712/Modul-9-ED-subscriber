# Conceptual Review

## 1. Apa itu AMQP?
AMQP adalah sebuah protokol komunikasi yang *provides* publisher-subscriber interface yang dapat memastikan pemindahan pesan secara cepat, aman, dan asinkronus (*by default*).

## 2. Apa maksud guest:guest@localhost:5672?
Guest yang pertama berupa username sedangka guest kedua berupa password yang terdefinisi saat rabbitmq pertama kali diunduh, lalu localhost:5672 adalah adress-port rabbitmq berada.
